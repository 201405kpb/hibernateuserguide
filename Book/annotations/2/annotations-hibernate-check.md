#### 24.2.9. `@Check`

<div class="paragraph">

The [`@Check`](https://docs.jboss.org/hibernate/orm/5.2/javadocs/org/hibernate/annotations/Check.html) annotation is used to specify an arbitrary SQL CHECK constraint which can be defined at the class level.

</div>
<div class="paragraph">
