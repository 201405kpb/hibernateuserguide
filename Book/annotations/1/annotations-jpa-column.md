 #### 24.1.9. `@Column`

    <div class="paragraph">

    The [`@Column`](http://docs.oracle.com/javaee/7/api/javax/persistence/Column.html) annotation is used to specify the mapping between a basic entity attribute and the database table column.

    </div>
